The project:

Contains World Chess Championship 2014 games replays
Magnus Carlsen vs Viswanathan Anand

Keys to use:
Space and Enter key, arrow keys.
Special keys: "/" - repeats last keypress
"C" - cursor mode on (can move pieces)
"N" - new game
"U" - undo mode
ESC - return

How to run ChessAtr program on Atari emulators:

----------------------------------------------------------------
 on Atari800win emulator
----------------------------------------------------------------

Menu > Input > Arrow keys > Set Control+arrows keyboard.

To open and run:
Menu > File > Load executable, open ChessAtr.xex file.
It loads in memory and starts OK.



----------------------------------------------------------------
 on Altirra emulator
----------------------------------------------------------------
Menu > Boot image (or Open image). Open ChessAtr.xex file.



How to compile:

Compile on MADS assembler for 8-bit CPU 8502 on Atari.
Prepare datas by using Lua programs.
